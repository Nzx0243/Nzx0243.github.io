## 二、 经典激活函数盘点

（在这个模块，逐一介绍最经典、最常用的函数。每个函数建议包含：公式、图像特点、优点、缺点。）

### 1. Sigmoid 函数

**数学公式**：

$$
\sigma(x) = \frac{1}{1 + e^{-x}}
$$

* **工作原理**：将输入映射到 (0, 1) 区间...
* **优点**：...
* **致命缺点（梯度消失）**：详细解释一下当 $x$ 特别大或特别小时，导数趋近于 0 的问题。

### 2. Tanh 函数 (双曲正切)

**数学公式**：

$$
\tanh(x) = \frac{e^x - e^{-x}}{e^x + e^{-x}}
$$

* **与 Sigmoid 的对比**：它是零中心化的（Zero-centered），通常收敛速度比 Sigmoid 快...

### 3. ReLU (Rectified Linear Unit)

**数学公式**：

$$
f(x) = \max(0, x)
$$

* **为什么 ReLU 能成为主流**：计算极快，缓解了正区间的梯度消失问题。
* **Dead ReLU 问题**：解释一下神经元“坏死”现象的原因。

## 五、 代码实战：在 PyTorch 中调用

（给读者展示一下实际写代码时怎么用，这在技术博客里非常受欢迎。）

```python
import torch
import torch.nn as nn

# 定义一个包含常见激活函数的简单网络层
class ActivationDemo(nn.Module):
    def __init__(self):
        super(ActivationDemo, self).__init__()
        self.relu = nn.ReLU()
        self.silu = nn.SiLU() # 即 Swish
        
    def forward(self, x):
        # x = self.relu(x)
        x = self.silu(x)
        return x
```
